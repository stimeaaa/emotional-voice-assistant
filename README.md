# Érzelmileg adaptív beszédalapú chatbot – Demonstrációs rendszer

Ez a repository a diplomatervemhez készült demonstrációs rendszer forráskódját tartalmazza. A rendszer egy érzelmileg adaptív, beszédalapú chatbot, amely elsőéves egyetemi hallgatók támogatását célozza meg szóban történő interakción keresztül.

A rendszer képes a felhasználó beszédének felismerésére, a hang alapján az érzelmi állapot és a beszédtempó becslésére, egy nagy nyelvi modell segítségével érzelmileg illeszkedő válasz generálására, valamint érzelmes beszédszintézisre.
